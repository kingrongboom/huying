给你的博客教程

发表于 2020-02-16 | 分类于 tutorial

我本打算，我自己写一个博客工具，到时候我们一起用的，甚至搞一个图形化的工具，但是，这个事情目前还在我的Plan，到达可用的目的需要较长的一段时间，这种情况下，我先出一个教程，让你用现有的工具。

用hexo

博客工具需要的环境

  1. node.js
  2. 中国npm镜像
  3. github

我们的整个过程都在macOS的终端中完成，请先打开你的终端（terminal）。

Git

首先，先检查你的git安装没有，一般都是有的。在终端输入git --version，如果没有提示command not found，说明这是没有问题的。

如果有问题，咨询黄先生。

hexo的使用

hexo的使用也是需要在终端完成的。

首先你找到一个文件夹，在这个文件夹下打开终端作为你的博客网站的根目录。

然后你运行hexo init初始化这个根目录。
